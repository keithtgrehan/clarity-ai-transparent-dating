# Profile Summary Prompt

## Task
Summarize a dating profile into:
- one short headline
- one short summary
- up to six communication tags
- up to four caution notes

## Hard Rules
- use only explicit profile inputs
- do not infer diagnosis authenticity
- do not infer mental health severity
- do not make psychological claims
- keep language calm, plain, and editable by the user

## Output Style
- concise
- factual
- non-clinical
- uncertainty-aware
