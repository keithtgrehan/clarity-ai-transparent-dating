# Moderation Review Process

## First-Pass Workflow
1. user report or AI-assist flag enters review queue
2. founder reviews structured metadata, message snippet, and context
3. founder classifies the issue and records action taken
4. block state is respected immediately when requested

## Review Questions
- what specific behavior crossed a platform boundary
- is there enough evidence to act
- does this require immediate hold or just follow-up review
- is the issue harassment, coercion, scam behavior, or another clear category

## Current Boundaries
- no fully automated permanent bans
- no clinical interpretation
- no diagnosis authenticity judgment

## Missing Operational Pieces
- moderator dashboard
- appeals workflow
- response-time SLA
- escalation rules for repeated offenders
