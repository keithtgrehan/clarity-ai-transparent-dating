# Onboarding Flow Notes

## UX Goal
Make users feel more understood through structure, not more analyzed.

## Flow Sections
1. relationship intent
2. communication preferences
3. sensory preferences
4. first-date comfort
5. profile preview and edit

## UX Rules
- one idea per section
- explicit labels over clever phrasing
- examples that reduce blank-page stress
- clear save states
- no urgency or streak language

## Copy Direction
- “What helps communication feel good?”
- “What kind of first date is easiest for you?”
- “How much planning helps you relax?”

## Risk To Watch
If onboarding feels like paperwork, shorten it before launch instead of hiding it behind AI magic.
